# Infographic & Slide Generator

> [!NOTE]
> このリポジトリは「DラボAIチャンネル」の学習用教材です。

テキスト要約からプレゼンテーションスライド（Marp形式）を作成し、さらに各スライドに対応する画像生成プロンプトを自動生成するためのプロジェクトです。

## 概要

このプロジェクトは、論文や長文テキストの要約から、視覚的なインフォグラフィックやスライド資料を効率的に作成することを目的としています。
AIエージェントのスキルやワークフローと組み合わせて使用することで、コンテンツ作成プロセスを自動化します。

## 主な機能

- **Marpスライド変換**: 手動またはAIワークフローにより、MarkdownテキストをMarp形式のスライドに変換します。
- **プロンプト自動生成**: `marp_to_prompts.js` スクリプトを使用して、Marpスライドを分割し、画像生成用の詳細なプロンプトファイル（Markdown）を生成します。
- **画像生成**: 生成されたプロンプトを用いて、AI（Gemini API等）によりスライド画像やインフォグラフィックを生成します。

## ディレクトリ構成

- `.agent/`: AIエージェント用のカスタムスキル (`image-gen`) やワークフロー (`texttoslide`) が格納されています。
- `scripts/`: 自動化のためのスクリプトが含まれています。
  - `marp_to_prompts.js`: Marpファイルを読み込み、スライドごとに分割して画像生成プロンプトを出力するNode.jsスクリプトです。
- `template/`: インフォグラフィックやスライドのデザインテンプレートが格納されています。
- `prompts/`: 生成された画像生成用のプロンプトファイルが保存されるディレクトリです。
- `outputs/`: 最終的な生成物が保存されるディレクトリ（想定）。

## セットアップ

このプロジェクトに含まれるスクリプトを実行するには、Node.js環境が必要です。

### 依存パッケージのインストール

`image-gen` スキルを使用するためには、専用のディレクトリで依存パッケージをインストールする必要があります。

```bash
cd .agent/skills/image-gen
npm install
```

### 環境変数の設定 (.env)

画像生成機能を利用するには、Gemini APIキーの設定が必要です。
プロジェクトのルートディレクトリに `.env` ファイルを作成し、以下の形式でAPIキーを設定してください。

```properties
GEMINI_API_KEY=your_api_key_here
```

`.env` ファイルは `.gitignore` に含まれているため、誤ってコミットされる心配はありません。

### 自動セットアップ

設定方法がよくわからない場合は、AntigravityやClaude Codeに対して**「このリポジトリを使えるように環境設定して」**と指示してください。必要なパッケージのインストールや環境設定のサポートを行います。

## 使用方法

### プロンプト生成スクリプトの実行

Marp形式のMarkdownファイルから、スライドごとの画像生成プロンプトを作成します。

```bash
node scripts/marp_to_prompts.js <入力ファイルパス>
```

**例:**
```bash
node scripts/marp_to_prompts.js ./path/to/slide.md
```

実行すると、入力ファイルと同じディレクトリにある `prompts` フォルダ内に、タイムスタンプ付きのフォルダが作成され、その中に各スライドのプロンプトファイル（Markdown）が生成されます。

### AIエージェント機能

このプロジェクトには、AIエージェント向けの拡張機能が含まれています。

- **/texttoslide**: テキストからスライド作成、プロンプト生成までを行うワークフロー。
- **image-genスキル**: 画像生成を実行するスキル。

## 開発ルール

- **文字コード**: すべてのテキストファイルは **UTF-8 (BOMなし)** で統一してください。
- **PowerShell**: Windows環境での操作はPowerShell 7 (pwsh) を推奨します。詳細は `AGENTS.md` を参照してください。
